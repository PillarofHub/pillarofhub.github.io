위의 링크를 클릭하면 해당 형식의 OMR 카드 페이지로 이동합니다.

언어이해, 휴식, 추리논증, 성적표 기능을 제공합니다. 또한 시험 종료 후 각 페이지에서 정답을 확인할 수 있습니다.

OMR 카드 오른쪽 위의 시작 버튼을 누른 시점부터 마킹이 가능합니다. 시간이 다 되면 자동으로 마킹이 중지됩니다.
